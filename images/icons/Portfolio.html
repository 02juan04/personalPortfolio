Card Order Memorization Game

This is a game where a user generates a randomized deck of card, memorizes the order, and tries to put the cards back in order

Main Menu

  • Input number of cards
  • Defaults to 52 if no input

Memorization Stage

  • User can Browse deck of cards by looking at the entire deck at once, or by going through a batch of three cards all the way till the end

Order Stage

  • When user is ready, they will see empty slots where they can input cards in order. Recieves feedback for right or wrong.

Task Tracker

This project allows the user to create, delete, and archive tasks.

New Tasks get added to the 'Tasks in Progress' list, while completed ones get added to the 'Completed Tasks' list.

Archiving tasks is also a feature. This allows users to put these tasks aside for later. These tasks get put into the 'Archived' tab, accessible through the navbar

Live Foot Ball Stats

This project utilizes API-Football for information about leagues, teams, and fixtures.

Users can choose from a variety of leagues, and view the current standings from users choice. Including: (but not limited to)

  • Champions League
  • Premier League
  • La Liga
  • Liga MX

Different seasons can also be chosen, limited by the information of the API

Standings Section

  • View up to Date standings for chosen league. Updates accordingly

League Matches Section

  • Displays last 10 matches of the league including LIVE matches
  • Dynamically changes UI if the game is Live, or Completed
